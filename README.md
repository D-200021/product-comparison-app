# Product Comparison App (React + TypeScript)

An interactive, responsive web application that allows users to compare multiple products side-by-side. Built using **React**, **TypeScript**, and **modern CSS**, this project demonstrates component-driven architecture, state management, accessibility, and user-centric UI design.

---

## Features

* **Product Listing**

  * Displays a grid of 8 products
  * Each product shows name, brand, image, price, and key features

* **Compare Products**

  * Select up to **3 products** to compare
  * Comparison view appears only when **2 or more** products are selected
  * Side-by-side comparison table

* **Smart Highlighting**

  * Lowest price highlighted
  * Highest battery capacity highlighted

* **Manage Comparison**

  * Remove individual products
  * Clear entire comparison list

* **Search & Filter**

  * Search products by name or brand

* **Persistence**

  * Selected comparison items are saved in `localStorage`
  * Data persists across page reloads

* **Accessibility**

  * Keyboard navigation supported (`Tab` + `Enter`)
  * Focus indicators for interactive elements

* **Light / Dark Mode**

* **Responsive Design**

  * Optimized for desktop and mobile screens

---

## Assumptions

* The application uses **static product data** (no backend or APIs)
* Data stored in `localStorage` is assumed to be valid and app-controlled
* Images are placeholders and can be replaced with real assets
